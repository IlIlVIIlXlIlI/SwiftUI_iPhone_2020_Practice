# SwiftUI_RunButton

### ボタンを押してテキストを更新する場合<br>
変数の最初に@Stateをつける<br>
@State var num = 0　のように<br>

@Stateを付けて宣言すると値の更新が監視されるようになり、<br>
値が変化するとViewも更新されるようになる<br>
